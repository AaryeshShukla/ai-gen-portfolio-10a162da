# Product Showcase

## Overview
This is a stunning product showcase page featuring a product image slider, interactive color and specs picker, customer reviews section, and a checkout widget.

## Features
- Bold visual hierarchy with large typography
- Custom product image slider
- Interactive color and specifications picker with animated previews
- Detailed technical specifications grid
- Interactive customer reviews section with star ratings
- Complete checkout widget with dynamic cart count and checkout modal

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. Enjoy the product showcase experience!

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts
- Flexbox and Grid Layouts
